# Code Rules

Conventions for writing and organising code. Language-specific rules are in subdirectories; files here apply across languages or to a specific toolchain.

---

## Files

| File | Covers |
|---|---|
| `asgard.md` | Asgard project conventions: repo structure, libraries, imports, logging |
| `bash.md` | Bash script style: preamble, variables, sections, terminal output |
| `naming.md` | Naming files and folders on disk |

## Subdirectories

| Directory | Covers |
|---|---|
| `python/` | Python language rules: layout, naming, functions, classes, errors, testing, setup |
